Using Make Video Tags
Send/Sync to iPod

After clicking the "Apply..." button, if one or more iPods are detected you will see the "Send to iPod" option:

Send/Sync to iPod

  • Click the "Send to iPod" checkbox.
  • Select the iPod you want to copy the tracks to from the popup.
  • If the selected iPod is set to manually manage then the tracks will be copied immediately.
  • If the selected iPod is set to auto-sync you can click the "Sync iPod immediately" checkbox to have the iPod sync right after the data is applied. Otherwise, files will be copied the next time the iPod syncs.
