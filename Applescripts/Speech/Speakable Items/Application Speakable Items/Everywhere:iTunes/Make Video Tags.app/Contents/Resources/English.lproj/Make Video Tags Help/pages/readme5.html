Using Make Video Tags
Step 5: Applying the Information

When you are satisfied with your data entry you can send the information from Make Video Tags to the selected tracks in iTunes by clicking the "Apply..." button or choose "Apply" from the File menu.

Only tag data that has been checkmarked will be applied to the selected tracks in iTunes. Comments, Description, Long Description and Lyrics info will be applied only if they differ from the current tag information.
