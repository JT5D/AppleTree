Using Make Video Tags
Opening Files

You may find it convenient to select some files in the Finder and have Make Video Tags add them to iTunes and then display them in its window for editing.

Activate Make Video Tags and select "Open Files..." from the File menu or press Command-O. Navigate to one or more files in the Finder and select them in the Open/Save dialog. Press the "Add to iTunes" button.

Make Video Tags will first add the files to iTunes. This can take a few minutes. Also, you may see the spinning beachball while files are being added. This is normal.

After the files have been added as tracks in iTunes, they will be loaded into Make Video Tags.

Adding to the batch:

If tracks are already loaded in Make Video Tags and you select additional tracks or files, you will be asked if you want to add the new tracks to the current batch of tracks or clear the current tracks without saving changes.
