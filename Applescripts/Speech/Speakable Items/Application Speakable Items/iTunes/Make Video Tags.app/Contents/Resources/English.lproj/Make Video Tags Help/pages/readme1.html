Using Make Video Tags
Step 1: Installation

The download package includes an application and an AppleScript both of which are named "Make Video Tags". The AppleScript named "Make Video Tags" allows you to activate the application named "Make Video Tags" from the iTunes Script menu.

If you installed the files from the downloaded disk image by dragging them to the "Installer" then the AppleScript and application have already been installed in your [user name] > Library > iTunes > Scripts folder. The script "Make Video Tags" will appear in iTunes' Script menu, where you can activate it by clicking on its name.

To manually install, if necessary: Put the AppleScript "Make Video Tags" in your iTunes Scripts folder. This is located in [user name] > Library > iTunes. If there is no folder named "Scripts" there, create one and put the script in it. It will then appear in iTunes' Script menu, and can be activated by selecting by name and clicking.

The application "Make Video Tags" can also be placed in the Scripts folder, or, if you prefer, your "Applications" folder.

IMPORTANT: If you are upgrading from a previous version of Make Video Tags make sure the earlier version is either deleted before installing or that the new files replace the old versions.
